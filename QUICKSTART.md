# Быстрый запуск

## Шаг 1: Запустить Backend

Откройте **новый терминал** и выполните:

```bash
cd backend
python -m pip install -r requirements.txt
python run.py
```

Вы должны увидеть:
```
INFO:     Uvicorn running on http://0.0.0.0:8000
```

**Оставьте этот терминал открытым!**

## Шаг 2: Запустить Frontend

В **другом терминале** (первый должен оставаться открытым):

```bash
cd frontend
npm install
npm run dev
```

Вы должны увидеть:
```
➜  Local:   http://localhost:3000/
```

## Шаг 3: Открыть в браузере

Откройте http://localhost:3000 в браузере.

## Важно!

- Backend должен быть запущен на порту 8000
- Frontend должен быть запущен на порту 3000
- Оба сервера должны работать одновременно

## Если видите пустой экран:

1. Откройте консоль браузера (F12)
2. Проверьте, нет ли ошибок
3. Убедитесь, что backend запущен (проверьте http://localhost:8000 - должно показать {"message":"Habits Tracker API"})
